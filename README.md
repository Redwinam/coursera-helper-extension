# Coursera 学习助手（Chrome 扩展）

提升 Coursera 学习体验的轻量级浏览器扩展，聚焦视频学习、内容整理与测验复盘三大场景：

- 视频页快捷控制：倍速一键切换、2 秒后退、截图复制到剪贴板（支持上下裁剪并记忆设置）
- 内容一键复制：复制页面标题（可选 HTML 格式）与字幕/阅读内容，自动清理时间戳与多余元素
- 测验助手：在反馈页保存作答数据，在尝试页加载并标注「之前正确/错误」，支持题目快捷复制

## 安装使用

1. 在 Chrome/Edge 打开「扩展程序」页面，开启右上角「开发者模式」
2. 点击「加载已解压的扩展程序」，选择本项目根目录
3. 打开 Coursera 课程页面（`coursera.org` 下的 `learn` 页面），右下角会出现控制面板

## 功能说明

### 视频学习
- 倍速按钮：预设 `1x ~ 3x`，当前选中高亮并立即生效
- 后退按钮：一键回退 2 秒，便于复盘关键讲解
- 画面复制：将当前视频画面复制到剪贴板（PNG），支持设置「上裁剪/下裁剪」，配置保存在本地

### 内容整理
- 复制标题：支持以纯文本或 HTML 结构复制页面标题
- 复制字幕/阅读内容：自动清理时间戳、按钮等冗余元素，阅读内容会自动在顶部附加标题
- 章节行内复制：在小节标题旁插入「复制」按钮，便于快速摘录

### 测验助手
- 保存答案（反馈页）：提取题干、选项、选择状态与正确性，并保存到本地
- 加载答案（尝试页）：根据保存记录自动勾选选项，并以样式标注之前正确/错误
- 题目快速复制：在每道题右侧提供「📋」按钮，支持一键复制题干与选项（A/B/C…）

## 权限与隐私

- 站点权限：`*://*.coursera.org/*`
- 浏览器权限：`storage`、`clipboardWrite`、`activeTab`、`scripting`
- 数据存储：使用 `chrome.storage.local` 保存测验数据与截图裁剪设置，仅保存在本地，不上传网络

## 项目结构

- `manifest.json` 扩展清单与权限声明
- `src/content/` 课程页功能模块（视频/内容/测验）与样式
- `src/popup/` 扩展弹窗开关，可独立启用/停用各模块

## 兼容性提示

Coursera 页面结构可能会不定期调整，如遇到按钮不出现或复制异常，欢迎在仓库 Issue 中反馈具体页面链接与复现步骤。

## 开发说明

- 本项目为「已解压扩展」，无需构建流程，修改文件后在扩展管理页点击「重新加载」即可
- 模块开关通过 `chrome.storage.local` 保存，弹窗中可随时切换
